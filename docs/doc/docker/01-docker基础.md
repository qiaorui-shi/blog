### docker基础指令
```
    docker version  # 查看docker版本
    docker info  # 查看docker信息
    docker images  # 查看本地镜像
    docker pull  # 拉取镜像
    docker run  # 运行镜像

    docker ps  # 查看运行的容器
    docker ps -a  # 查看所有的容器
    docker stop  # 停止容器
    docker start  # 启动容器
    docker restart  # 重启容器
    docker rm  # 删除容器
    docker rmi  # 删除镜像
    docker logs  # 查看容器日志
    docker exec  # 进入容器
```
